회원 가입 - 이메일 형식 필터링
( 000@000.000 타입의 이메일 형식이 아니면 필터링 되는 기능)

회원 가입 - 이메일 인증 후 메일 주소 변경 방지
(인증 완료 후에 이메일을 수정해도 필터링 되는 기능)

매장 검색 - join 중복 데이터 필터링
데이터 등록된 StoreId로 매칭을 한 후에 같은 StoreId의 데이터(중복되는 데이터)를 Group by로 묶어서 한 번에 가져옴

온라인 예약 - 시스템
매장에서 지정한 인원, 시간대(영업시간) + 휴무일에 맞춰 가능한 예약의 목록을 리스트업.

온라인 예약 - 날짜 필터링
강제로 비활성화 된 날짜로 설정하여 예약을 시도해도 필터링 되는 기능

리뷰 등록 - 이미지 업로드 + 수정
구체적인 로직 설명(드래그 순간부터 등록 전 로직, 등록 후 로직 등)

맛집 로드맵 - 위치 정보 제공 거절 시 예외 처리
위치 정보를 제공 받지 못 할 경우 보이는 기본 주소 설정 및 코드 구현\

맛집 로드맵 - 마커 오버레이
리스트를 불러올 때 리스트 안에 가게, 주소 정보를 넣어두고 클릭했을 때 주소를 좌표로 변환하여 지오코더를 활용하여 위치 검색
리스트, 지도의 마커를 클릭하면 해당 위치를 중심으로 하며, 매장 정보 오버레이가 출력됨

카테고리 관리 
리스트에 변화가 일어났을 때(등록/수정/삭제 등) 선택되는 카테고리의 기준
등록 수정 = 기존에 선택했던 카테고리
삭제 = 1번째 카테고리

사업장 관리 - 예약/휴무일 파트
예약-휴무일 관의 연관성 위주
